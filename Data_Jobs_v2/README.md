# Data Jobs Dashboard v2 w/ Power Bi

![Dashboard Page 1](/images/project2_page1.jpg)

## Introduction

This dashboard (V2.0) provides a clear, unified view of the 2024 data job market, designed to support job seekers in understanding role demand, salary ranges, locations, and employment types. The report, which was created using a real-world dataset of data science job postings, focuses on providing critical market insights through a simple, single-page design.

## Skills

- **Data Transformation (ETL) with Power Query**: Cleaned and prepared raw data by handling blanks, correcting data types, and creating derived columns.
- **Data Modeling**: Built efficient data models using star schema principles and defined relationships.
- **DAX Measures & Calculations**: Created measures to calculate KPIs such as job count, skills per job, and median salaries.
- **Visualisations Used**:
  - **Core Charts**: Utilised Column, Bar, Line and Area Charts to compare job counts and track trends over time.
  - **KPI Indicators & Tables**: High-level metrics and detailed breakdowns
- **Interactive Reporting**:
  - **Slicers**: Enabling dynamic, user-driven data filtering.
  - **Buttons & Bookmarks**: For streamlined navigation and managing report views.

## Dashboard Overview

This second iteration improves on the original version by prioritising simplicity, faster insight discovery, and a more focused single-page layout, designed to give job seekers the most critical market insights.

![Dashboard Page 1](/images/project2_page1.jpg)

This page showcases key performance indicators (KPIs) like Job Count, Skills Per Job, Median Yearly Salary, and Median Hourly Salary. You can also quickly see Skill Popularity (by job percent or count) and compare Salaries across different Job Titles, all designed for an efficient overview.

## Conclusion

This dashboard (V2.0) presents a concise and practical view of the 2024 data job market, designed to help job seekers quickly understand role demand, salary benchmarks, and skill expectations. By consolidating key insights into a single-page report, the dashboard balances clarity with depth, allowing users to explore the market efficiently without unnecessary complexity. This iteration strengthened my skills in data modelling, DAX calculations, and Power BI report design, with a clear focus on usability and insight-driven storytelling. The project reflects how analytics can be used to support real-world decision-making rather than simply visualising data.
